# LinkedIn Profile Analyzer

A simple Python tool that analyzes your LinkedIn headline and about section and gives feedback.

## Features

- Profile score
- Keyword detection
- Optimization suggestions
- Simple UI

## Run Locally

Install dependencies

pip install -r requirements.txt

Run the app

streamlit run app.py

## Demo

Paste your LinkedIn headline and about section to analyze your profile.
